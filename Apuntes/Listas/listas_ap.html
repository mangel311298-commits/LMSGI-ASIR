• <form>: Contenedor principal del formulario. Usa action y method.
  • <label>: Etiqueta descriptiva para cada campo. Mejora accesibilidad.
  • <input>: Campo de entrada. Tipos comunes: text, email, password, number, submit, reset, radio, checkbox, hidden.
  • <select> y <option>: Menús desplegables para elegir entre varias opciones.
  • <textarea>: Área de texto para comentarios largos.
  • <fieldset> y <legend>: Agrupan campos relacionados con un título.
  • Validación básica: Usa required, maxlength, pattern, min, max.
  • Accesibilidad: Relaciona <label> con id del campo. Usa aria-label si es necesario.
  • Diseño y estilo: Aplica CSS para espaciado (margin, padding) y responsividad (flexbox, grid).
  • Seguridad: Valida también en el servidor. Usa method="post" para datos sensibles.
  • Campos modernos: type="date", color, range, file, search.
  • Botones: <input type="submit">, <input type="reset"> o <button> personalizado.
